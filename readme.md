# Vite React Template

> 兼容性:完全放弃 IE

## tailwind

使用 [tailwindcss@3](https://tailwindcss.com/docs/installation)

> 错误 `Unknown at rule @apply`
>
> 解决方法：编辑器安装扩展: `PostCSS Language Support`

### 与 antd@4 结合

可以与 antd 同时使用, 并实现按需加载.

## react router v6

本例中, 使用 [useRoutes](https://reactrouterdotcom.fly.dev/docs/en/v6/upgrading/v5#use-useroutes-instead-of-react-router-config) 来配置路由, 并结合了权限校验和懒加载.

## Recoil

把 mobx 换成 [Recoil](https://recoiljs.org/zh-hans/)
